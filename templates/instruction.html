Documentation of PhoneScanner

Simple tool to check Android or iOS phones for IPS-spyware. Code: GitHub

Prepare the phone for scanning

Android Phones
For Android device, the developer options on the phone needs to be activated. Developer options provide functionality for PhoneScanner to scan the device. The exact steps might vary from phone manufacturer and version of Android, but roughly the following steps will help you activate4 developer options and USB debugging. PhoneScanner will try to deactivate the developer mode at the end of the scan. But for some phone this procedure false, so if you think that's a problem, then please don't turn it on, or scan your device.
  1. Go to Settings. Either find the Settings app in the phone drawer, or pull down from the top notification bar, on the top right you will see a gear type icon (⚙) for settings.
  2. Scroll down to find About Phone, search for Build number, tap on the build number 6-8 times to activate the developer mode. For some phones, Build number might be hidden under Software info. (Warning: For most phone we can turn it off (see step 4), but there are some phones this cannot be undone. This will not interfere with any of the phone’s common functionality, but does not leave the phone at the state as it was before the scanning.)
  3. Next step is to turning on USB debugging. Go to Settings→Developer options→USB debugging. Tap on the toggle switch to turn it on.
  4. Turning off developer options: The scanner will automatically try to reset the developer options once Save or Reset is pressed. Restart the phone to get to original state. If you still see the developer options (a.k.a, developer mode) is on, open Settings→Apps, you will see list of apps installed in your device. Find the app Settings in list of apps, tap on it, and tap clear data (sometimes clear data option is listed inside Storage). This should clean the developer options. [Warning: For some Android, phone this might fail. So please check the settings again. developer options , but not all.]
iOS/iPhones
There is not much preparation required for iOS, just connect the USB cable, on the device you should see a prompt asking for the phone passcode (also call pin) number. The passcode has to be entered to unlock the device, and trust the computer to allow scan the iPhone. Troubleshooting: Sometimes the PhoneScanner might fail to recognize the iPhone. Try opening iTunes on the MAC, and see if the phone is listed. Disconnect the phone from the USB cable, reconnect, and try scanning again.
